# 20260424 图文解析保存到知识库执行计划

## 目标
在图文复刻/解析面板中，为解析结果增加“保存到文件夹”能力，支持将解析后的文案写入现有知识库文件夹，作为后续知识库检索或小红书图文排版素材。

## 范围
- `app/(main)/scripts/ImageTextReplicationPanel.tsx`
- `app/api/knowledge/folders/*`
- `lib/knowledge.ts`（如需要复用路径/文本处理）
- 文档索引 `docs/README.md`

## 里程碑
1. 复用知识库文件夹/文件接口，确认写入格式
2. 在解析结果旁新增保存入口
3. 完成保存后的反馈与错误处理
4. 本地 lint/typecheck 验证

## 风险
- 解析文本过长导致知识文件切分过细或写入失败
- 文件夹选择不明确导致保存到错误位置
- 现有知识库 schema 未初始化时需要友好提示

## 回滚
- 仅移除新增保存入口和相关 API 调用，不影响现有解析与生成流程

## 验收标准
- 解析结果可一键保存到指定文件夹
- 保存后在知识库中可检索/查看
- 保存失败有明确提示
- `npm run lint`、`npm run typecheck` 通过
