# 20260511 小程序智能复刻音频、下载与分镜对比修复计划

## 目标

修复小程序智能复刻链路中的三个体验问题：

1. 第三阶段生成视频在详情页播放时无声
2. 第三阶段生成视频缺少下载入口
3. 产品/角色替换页缺少生成分镜网格图与原始分镜网格图的对比切换

## 范围

- `digital_human_miniapp/taro/src/subpages/remix-video-detail/`
- `digital_human_miniapp/taro/src/subpages/storyboard-board/`
- 必要时补充作品/任务状态相关只读逻辑排查

## 方案对比

### 方案 A：只改前端展示层

- 优点：改动小，上线快
- 风险：如果视频源本身无音轨，仅放开静音按钮无法真正解决

### 方案 B：前端播放链路修复 + 产物地址选择排查

- 优点：同时覆盖播放器参数、选中视频地址与素材历史切换问题
- 风险：需要额外联调确认是否存在上游静音产物

结论：采用方案 B。

## 兼容性结论

- Taro `Video` 组件支持 `muted`、`showMuteBtn`、`enablePlayGesture` 等能力，可兼容当前小程序播放器交互
- 分镜网格图切换仅基于现有任务数据字段，不引入新的接口协议
- 下载能力复用现有代理下载与相册权限申请逻辑，兼容当前 API 代理体系

## 风险

- 部分历史生成视频可能确实没有音轨，前端只能确保不误静音，不能为静音视频补声
- 分镜网格图原图字段来源存在多种兜底路径，需要谨慎回退，避免切换后空白

## 回滚策略

- 若详情页播放器修复引入回归，可回退到上一版播放器参数
- 若网格切换字段命中不稳定，可保留生成图展示并临时隐藏切换按钮

## POC / 不确定点

- 通过 weapp-dev-mcp 打开复刻详情页，验证播放器取消默认静音后是否可正常出声
- 通过真实复刻任务验证原图/生成图切换是否都能正确展示

## 里程碑

1. 计划文档落库并登记索引
2. 完成详情页下载与播放器修复
3. 完成替换页分镜网格切换

## 验收标准

- 第三阶段详情页存在明确下载按钮，保存成功
- 有音轨的视频在详情页播放可正常出声
- 产品/角色替换页可在原图与生成图之间切换预览
- 小程序控制台无新增异常日志
